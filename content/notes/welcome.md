## 为什么要创建这个博客

作为一名开发者，我一直想要有一个属于自己的空间，用来记录学习过程中的点滴，分享自己的开源项目和技术经验。

这个博客采用了现代简约的中国风设计，使用 Next.js 和 React 构建，部署在 GitHub Pages 上。

## 博客的特点

- 现代简约的中国风设计
- 基于 Markdown 的内容管理
- 完全静态，无需后端
- 响应式布局，适配各种设备

## 未来规划

接下来我会持续更新博客内容，包括：

- 技术学习笔记
- 开源项目介绍
- 编程经验分享

希望这个小小的博客能够帮助到同样在学习路上的你。
