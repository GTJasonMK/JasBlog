## 项目简介

JasBlog 是一个采用现代简约中国风设计的个人博客系统。它基于 Next.js 构建，支持静态导出部署到 GitHub Pages。

## 主要特性

### 中国风设计

- 朱砂红为主色调的配色方案
- 云纹装饰和印章效果
- 思源宋体等中文字体支持

### 内容管理

- 基于 Markdown 的文章管理
- 支持 frontmatter 元数据
- 标签分类和筛选

### 交互体验

- 打字机效果的首页标语
- 阅读进度条
- 平滑滚动的返回顶部
- 文章目录导航

## 技术实现

项目采用了现代前端技术栈，确保良好的开发体验和运行性能。

## 部署方式

支持一键部署到 GitHub Pages，也可以部署到 Vercel、Netlify 等平台。
