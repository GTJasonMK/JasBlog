这是一个示例知识图谱，展示了如何组织和可视化知识点之间的关系。

## 图谱说明

- **示例知识点**：根节点，代表核心主题
- **子主题 A**：第一个分支，可以是某个概念或领域
- **子主题 B**：第二个分支，与主题相关的另一个方向

## 使用场景

知识图谱适合用于：

1. 学习路径规划
2. 概念关系梳理
3. 知识体系构建

```graph
{
  "nodes": [
    {
      "id": "1",
      "type": "default",
      "position": { "x": 250, "y": 50 },
      "data": { "label": "示例知识点" }
    },
    {
      "id": "2",
      "type": "default",
      "position": { "x": 100, "y": 150 },
      "data": { "label": "子主题 A" }
    },
    {
      "id": "3",
      "type": "default",
      "position": { "x": 400, "y": 150 },

    { "id": "e1-2", "source": "1", "target": "2" },
    { "id": "e1-3", "source": "1", "target": "3" }
  ]
}
```
